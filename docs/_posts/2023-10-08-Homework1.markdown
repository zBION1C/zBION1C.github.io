## Theory

### What is statistic and what are its relationships with other disciplines? What are some example of use in cybersecurity?
  
  > Statistic is the collection, analysis and presentation of data for the study and prediction of some phenomena.
  > It is related to a lot of scientific disciplines, for example:
  > - the study of geological phenomena, like how the temperature of the planet changed in the years
  > - In sociology statistics is used to study human behaviour.
  
  > In cybersecurity one application of statistics is **anomaly detection**, where we model the normal behaviour of a system via statistics means, and if the behaviour deviates from what we expect, then there is an anomaly.

### Describe the concepts of variable, attribute, population, sample and dataset?

The population in statistics is a set of statistical units on which i want to study something.

A population sample is a subset of a larger population. This sample has to be representative of the whole population, so it must be choosen at random.

An attribute is an abstract concept that i want to study on a population or a sample.

A variable is the ‘concretization’ of an attribute in the study. It has some values that it can take and this values are called level of measurement. Variables that have different level of measurement are different.

A dataset is a matrix on wich the rows represent the statistical units of the population, and for each row we have some variables associated to the corresponding statistical unit.

### Briefly describe the main sampling techniques?

Simple random sampling, cluster sampling, accidental sampling.
[source](https://en.wikipedia.org/wiki/Sampling_(statistics))

### Briefly describe the main experiment designs?

Independent measures, repeated measures, matched pairs.
[source](https://www.simplypsychology.org/experimental-designs.html)

The following javascript code draws a point, a line, a circle and a rectangle in a canvas element:

<script>
    var canvas = document.createElement('canvas');
    var ctx = canvas.getContext('2d');
    ctx.canvas.width = window.innerWidth;
    ctx.canvas.height = window.innerHeight;


    document.body.appendChild(canvas);
    
    function drawPoint() {
        if (canvas.getContext) {
            var X = canvas.width / 3;
            var Y = canvas.height / 2;
            var R = 2;
            ctx.beginPath();
            ctx.arc(X, Y, R, 0, 2 * Math.PI, false);
            ctx.lineWidth = 3;
            ctx.fillStyle = "black";  // color inside the circle
            ctx.strokeStyle = 'black';  // color of outline
            ctx.fill(); // fill inside the circle
            ctx.stroke() // outline
        }
    }

    function drawLine() {
        if (canvas.getContext) {
            var X = canvas.width / 2.5;
            var Y = canvas.height / 2;
            ctx.beginPath();
            ctx.moveTo(X, Y);
            ctx.lineTo(X+20, Y);
            ctx.strokeStyle = 'black';  // color of outline
            ctx.lineWidth = 3;
            ctx.stroke(); // outline
        }
    }

    function drawCircle() {
        if (canvas.getContext) {
            var X = canvas.width / 1.7;
            var Y = canvas.height / 2;
            var R = 25
            ctx.beginPath();
            ctx.arc(X, Y, R, 0, 2 * Math.PI, false);
            ctx.lineWidth = 2;
            ctx.fillStyle = "orange";
            ctx.strokeStyle = "orange";
            ctx.fill();
            ctx.stroke();
        }
    }

    function drawRect() {
        if (canvas.getContext) {
            var X = canvas.width / 1.4;
            var Y = canvas.height / 3;
            ctx.beginPath();
            ctx.rect(X, Y, 70,50);
            ctx.stroke();
        }
    }

    drawPoint();
    drawLine();
    drawCircle();
    drawRect();
</script>
